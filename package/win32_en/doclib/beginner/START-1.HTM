step1: Let's check the working folder

First, let's open a folder where you can work on your own. In HSP, the folder to handle first is called "My Documents". "My Documents" can be opened from the Start menu or Explorer. You may have a file that you are using elsewhere, but that's okay. First, let's copy the files needed for this introduction.


HSP3TTL.JPG: Image files used in HSP demo

This file is located in "doclib" under the folder where HSP is installed. If you find the "HSP3TTL.JPG" file, copy it into the first opened "My Documents". (Be careful not to move the files.) Now, the working folder is ready. Up to this point, we are still in the preparation stage.
