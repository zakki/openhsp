Beginning Beginning ~ ~

Now, this is the beginning of the HSP programming course. Programming sounds difficult, but in reality, you just write the instructions written in English in the order in which you work. No, I don't know about other language programs ... but with HSP, it is. Let's learn comfortably without thinking too much.

Course flow

This course will be divided into the following parts. It is a good idea to proceed while actually checking the examples explained on HSP. If you have any questions, it is important to think about it and try it for yourself. Please remember while understanding step by step.

  1. Let's check the working folder
  2. Let's use the HSP script editor
  3. Super easy script
  4. Let's issue a command
  5. Flag the script
  6. Is this the launcher ?!
  7. Let's use variables
  8. More advanced condition judgment
  9. The road that continues

If you read and master all of these, you will be able to play an active role as an HSP programmer.
