The road that continues

How was it ?? This document covers everything from setting up HSP to the basics of writing scripts. Writing scripts, of course, isn't all that's described here. However, if you keep the basics described here, even beginners should be able to make some software. And for those who want to step up further, please read HSPPROG.HTM and HSPREF.HTM, which have all the HSP scripts. Don't be afraid to write scripts as much as you can.

Finally from the author

Much of what is described in this document, the concept of variables and labels is similar to other languages, especially BASIC. I wrote this document, hoping that people who are new to HSP will find it useful to know about programs when it comes time to move to Visual Basic, Delphi, C, etc. I hope HSP will increase the number of people who know the joy of programming (^^)
