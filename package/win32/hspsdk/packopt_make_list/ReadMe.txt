(1). 当アプリの概要

    実行ファイル自動作成での#packoptをリスト化するのを補助します。



(2). #packoptについて

    #packoptについて詳しくは下記のWebページをご覧ください。

    「プログラミング・マニュアル〜2.9.実行ファイルの作成方法」
    https://www.onionsoft.net/hsp/v37/doclib/hspprog.htm#MAKE_EXE



(3). 使い方

   1. 必要な項目に情報を入力してください。

   2. 「バージョンリソース設定(ファイル名)」を設定するには、「新規」ボタンをクリックすると、別ウィンドウが開きますのでそちらからバージョンファイルを作成してください。

   3. 「#packoptリスト作成」ボタンをクリックします。

   4. テキストボックスに#packoptのリストが作成されますので、「標準HSPスクリプトエディタのカーソルの位置に貼り付け」もしくは「クリップボードにコピー」ボタンをクリックしてコピーしてください。

   5. 終わり。
