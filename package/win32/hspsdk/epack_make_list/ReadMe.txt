(1). 当アプリの概要

    実行ファイル自動作成での#pack/#epackをリスト化するのを補助します。



(2). 使い方

   1. 「新規追加」ボタンをクリックしてプログラム名を登録してください。
   2. ウィンドウ左側のテキストボックスに、リスト化したいファイルの拡張子のみ("."ドットは不要)を入力します。
   3. 次に「フォルダ選択」ボタンをクリックして、リスト化したいフォルダを選択します。
   4. 「リスト作成」ボタンをクリックすると中央のテキストボックスに #packもしくは#epack のファイル名が羅列されます。
   5. 「クリップボードにCOPY」ボタンでクリップボードにコピーするか、もしくは「ファイとして保存」ボタンでファイルとして保存してください。
   6. 終わり。
