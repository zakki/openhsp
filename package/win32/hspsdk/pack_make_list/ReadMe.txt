(1). 当アプリの概要

    実行ファイル自動作成での#pack/#epackをリスト化するのを補助します。



(2). 使い方

    1. 「新規追加」ボタンをクリックしてプログラム名を登録してください。

    2. ウィンドウ左側のテキストボックスに、リスト化したいファイルの拡張子のみ("."ドットは不要)を入力します。

    3. 次に「検索フォルダ選択」ボタンをクリックして、リスト化したいフォルダを選択します。

    4. サブフォルダも検索対象としたい場合は、検索したいフォルダ名をリストボックスからクリックして選択してください。
       サブフォルダ内にさらにサブフォルダがある場合、自動的にこのサブフォルダも検索対象となります。

    5. 「リスト作成(+設定保存)」ボタンをクリックすると中央のテキストボックスに #packもしくは#epack のファイル名が羅列されます。

    6. 「クリップボードにCOPY」ボタンでクリップボードにコピーするか、もしくは「ファイルとして保存」ボタンでファイルとして保存してください。

    7. 終わり。



(3). 「DPMファイル作成」について

    「検索フォルダ選択」で抽出したファイルリストがDPMファイルの対象となります。
