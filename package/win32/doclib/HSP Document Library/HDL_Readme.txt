■ 概要
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP Document Library (HDL) は、HSP のドキュメントをすべてまとめて 検索・
閲覧 するためのブラウザです。

hs ヘルプファイル、txt/HTML ドキュメント、サンプルスクリプトについて、
横断検索や相互的な参照をすることができます。



■ インストール方法
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP Document Library を HSP のヘルプとして使用するためには、アーカイブ
内容のファイルとディレクトリを HSP インストールディレクトリへ上書きコ
ピーします。

hsphelp\helpman.exe は、ワンキーヘルプに連動して HDL を起動するための
ダミープログラムとなっています。ワンキーヘルプのプログラムを切り替えら
れるエディタを使用している場合は、ヘルププログラムとして hdl.exe を直
接指定してください。



■ HSP Document Library の使い方
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
ワンキーヘルプを呼び出すか、hdl.exe を起動すると、ドキュメント検索用の
データベースが自動的に作成され、HDL が使用可能になります。文字コードの
問題で読み出せないファイルがある場合は、データベース作成時にエラーダイ
アログで通知されます。

HDL の使い方や動作については、hdl_usage.htm を参照してください。

hs ファイルの書式は、HS_BIBLE.txt を、ドキュメント作成時の推奨事項など
は hdl_docrule.htm を参照してください。



■ 動作環境について
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HDL は、Windows 95, Windows NT 4.0 およびそれ以前のバージョンの 
Windows では動作しません。

初期バージョンの Windows 98 (+ IE4.0) で動作させるためには、OS コンポ
ーネントの更新が必要となります。

インストールするコンポーネント
・DCOM98 (http://www.microsoft.com/japan/com/dcom/dcom98/dcom13.aspx)
・atl.dll (http://activex.microsoft.com/controls/vc/atl.cab)

Internet Explorer セキュリティ強化の構成 (Windows Server) が有効になっ
ている環境では、一部の動作が制限される場合があります。



■ アンインストール
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HDL によってコンピュータに保存された設定ファイル、データファイルを消去
する場合は、同梱の hdl_uninst.bat を実行してください。

HDL をアンインストールする場合は、hdl_uninst.bat を実行後、インストー
ル時にコピーしたファイルを削除し、HSP 本体オリジナルのファイルを復元し
てください。



■ 著作権フリー
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP Document Library は、著作権フリー (パブリックドメイン) です。
