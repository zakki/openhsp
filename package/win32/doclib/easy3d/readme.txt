http://witchcraft-jp.sakura.ne.jp/

インストール方法
　e3dhsp3.dllとE3D_EXTフォルダを、hsp3フォルダにコピーしてください。
  e3dhsp3.asをhsp3フォルダの中のcommonフォルダの中にコピーしてください。
  exeを作成して使用する場合は、exeと同じフォルダにも
　e3dhsp3.dllとE3D_EXTを置いてください。
