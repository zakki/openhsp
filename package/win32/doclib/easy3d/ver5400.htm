ver5.4.0.0での変更点


ver5.4.0.0からVC2010ランタイムが必要になります。
Visual Studio (C++) 2010をインストールしていない人は
vcredistをインストールしてください。

32bitOSの場合は、ここから最新のVC2010ランタイムをダウンロードし、インストールします。
64bitOSの場合は、ここから最新のVC2010ランタイムをダウンロードし、インストールします。

管理者権限でインストールしてください。

１回インストールすればそれ以降のexe実行は必要ありません。

このVCランタイムは開発者だけでなくゲームプレイヤもインストールする必要があります。
ゲームを配布する際にはvcredistのダウンロード場所をreadme.txtに書くか
またはvcredistのexeを同梱してください。
